# Brainstorming Command

Run an interactive brainstorming session. Use this when you need to explore multiple approaches, generate creative ideas, or structure complex thinking.

## Process
1. Start with the problem or topic provided
2. Ask clarifying questions if needed
3. Generate multiple distinct approaches
4. Evaluate tradeoffs for each
5. Summarize key insights and next steps

## Questions
- What are the constraints?
- Who are the stakeholders?
- What are 3-5 alternative approaches?
- What are the pros/cons of each?
- Which has the best cost/benefit ratio?
- What would success look like?
